# 兰州财经大学光伏潜力评估平台 - 使用说明

## 项目概述

本项目是兰州财经大学周边公共建筑屋顶光伏潜力评估与碳减排决策平台的前端实现。平台采用现代化的技术栈，提供直观的数据可视化展示，帮助用户了解校园周边建筑的光伏潜力和碳减排效果。

## 技术栈

| 技术 | 版本 | 说明 |
|------|------|------|
| Vue | 3.4.x | 前端框架 |
| Vite | 5.2.x | 构建工具 |
| Element Plus | 2.6.x | UI组件库 |
| ECharts | 5.5.x | 图表库 |
| SCSS | 1.72.x | 样式预处理 |

## 快速开始

### 安装依赖

```bash
npm install
```

### 开发模式

```bash
npm run dev
```

启动后访问 http://localhost:5173

### 生产构建

```bash
npm run build
```

构建产物输出到 `dist` 目录

## 功能模块说明

### 1. 顶部导航栏 (Header)

包含以下功能：
- 兰州财经大学校徽和平台名称
- 实时天气信息显示（温度、天气状况、风力）
- 实时时钟（自动更新）
- 管理员用户信息展示

### 2. 左侧导航菜单 (Sidebar)

提供8个功能模块入口：
- 首页 🏠
- 适宜性评估 📊
- 效益评估 💰
- 碳减排评估 🌿
- 决策支持 🎯
- 可视化分析 📈
- 运维管理 ⚙️
- 系统管理 🔧

### 3. 主内容区域 (MainContent)

#### 地图展示区
- 搜索框（支持建筑名称或地址搜索）
- 图层切换工具（三维场景、影像底图、热力图、行政区划）
- 校园区域标记点

#### 统计卡片
展示6项核心指标：
- 研究范围（35.6万m²）
- 建筑总数（1,258栋）
- 可安装面积（67.28万m²）
- 年发电量潜力（3,186.76万kWh）
- 年减碳量（1.12万t）
- 投资回收期（6.35年）

#### 评估流程
展示6个评估步骤：
1. 数据准备 → 数据收集与处理
2. 适宜性分析 → 多因子空间分析
3. 效益评估 → 发电量与经济效益
4. 碳减排评估 → 碳减排量核算
5. 决策支持 → 综合评价与优化
6. 结果输出 → 可视化与报告生成

#### 数据图表
- **发电量趋势分析**：折线图展示月度预测与实际发电量对比
- **适宜性分级统计**：环形图展示不同适宜等级的面积分布
- **碳减排贡献TOP5**：横向条形图展示各建筑碳减排排名

#### 光伏潜力概览
展示6项潜力指标：适宜面积、装机容量潜力、年发电量潜力、年减碳量、投资估算、年收益

### 4. 右侧面板 (SidePanel)

#### 平台公告
展示最新公告列表，支持点击查看详情

#### 快捷入口
6个快捷功能按钮：适宜性评估、效益评估、碳减排评估、三维场景、专题制作、报告导出

#### 数据统计
饼图展示各建筑类型占比（体育馆、图书馆、教学楼、学生公寓、实验楼、其他）

#### 三维场景预览
校园三维场景预览图，支持点击进入全屏预览

## 常见问题

### Q1: 图表不显示怎么办？

A: 检查浏览器控制台是否有错误日志。确保：
- ECharts库已正确安装
- 图表容器元素已挂载到DOM
- 容器有明确的高度设置

### Q2: 开发服务器无法启动？

A: 检查端口是否被占用，或尝试修改 `vite.config.js` 中的端口配置。

### Q3: 样式显示异常？

A: 确保SCSS依赖已安装，并检查样式文件路径配置是否正确。

## 开发注意事项

1. **环境变量**：如需配置API地址等环境变量，可在项目根目录创建 `.env` 文件
2. **代码规范**：建议使用ESLint进行代码检查
3. **性能优化**：图表初始化已添加防抖处理，窗口resize时会重新渲染
4. **浏览器兼容性**：建议使用现代浏览器（Chrome、Firefox、Edge）
